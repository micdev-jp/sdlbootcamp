## 0.2 LOG取得
- sshでログイン確認
```
$ ssh root@10.0.0.1
$ cd /var/log
$ tail sdlc.log
```
* scpでログファイルを取得する
```
$ scp root@10.0.0.1:/var/log/sdlc.log ./
```

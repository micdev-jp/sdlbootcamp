0.2 ブート用SDカード作成方法¶

  • イメージをダウンロード
    https://github.com/micdev-jp/sdlbootcamp/tree/master/img

  • SDカードに焼く (SDカードが /dev/sdx にマウントされている場合)

$ gzip -dc < sdlbootcamp.img.gz | sudo dd of=/dev/sdx bs=4M
$ sync
